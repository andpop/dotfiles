# Editor Vim
* Use `<C--[>` instead of `<Esc>`
* `<leader>` – Space
## Plugins
* `:PlugInstall` — plugin installations
## File Tree
* `<leader><leader>t` — toggle file tree panel
## Navigation
## Editing
### Normal mode
* `[<leader` and `]<leader>` — add empty line above and under cursor (in normal mode)
### Insert mode
* `<C-K>M-` — Em dash
* `<C-K>N-` — En dash
